Скрипт для добавления данных из файла формата csv в базу данных.

Файл dataBaseConnect.xml содержит шаблон пользовательских конфигурации для доступа к БД
и имя таблицы, в которую заносятся данные (если таблица не существует - создается новая,
по умолчанию все поля имеют тип данных varchar(255)).

Файл index.php содержит форму для загрузки csv файла. На основании первой строки csv
файла и имени таблицы, которое прописано в пользовательских конфигурациях создает
(если не существует) таблицу с таким именем и именами полей, взятыми из первой строки.
Файл csv обрабатывается и заносится в базу данных, после осуществляется вывод из
базы данных в таблицу.
(присутсвует так же вывод имён полей таблицы с первой строки csv файла)
